### Вимоги до реалізації
- Вимога до Git: завдання 2 має бути зроблене окремим комітом (рекомендовано назвати коміт: Add JavaFX GUI for UUID search) або винесене в окрему гілку "gui" у репозиторії, щоб чітко відділявся GUI-шар від основної логіки.

- Додайте JavaFX до Maven-проєкту.
- Підключіть залежності JavaFX та налаштуйте запуск GUI через Maven (javafx plugin).
- Додайте клас-стартер GUI.
- Створіть окремий клас, який запускає JavaFX (наприклад, MainApp extends Application).
- Додайте форму для наповнення колекції.
- У GUI має бути можливість створювати/додавати об’єкти у вашу колекцію:
  - поля введення для основних атрибутів (мінімум 2–3 поля + тип об’єкта, якщо у вас ієрархія);
  - кнопка “Додати”;
  - після додавання об’єкт отримує UUID автоматично.
  - Додайте поле/блок для виводу всієї колекції (скорочено).
- Потрібно показати список у форматі “коротко”, наприклад:
  - назва/ключовий атрибут + UUID;
без повного переліку всіх полів.
  -Наприклад: Phone: iPhone 15 | UUID: ...

- Додайте пошук за UUID через GUI.
- У GUI має бути:
  - поле вводу UUID;
  - кнопка “Знайти”;
  - вивід повної інформації по знайденому об’єкту (всі поля + UUID) в окремому блоці. 
  - Якщо UUID введено некоректно — показати повідомлення про помилку. Якщо об’єкт не знайдено — показати повідомлення “не знайдено”.
### Як запустити додаток
1. Клонувати репозиторій, а саме гілку *gui*
```code
git clone -b gui --single-branch git@github.com:BohdanSharubin/java-university-project.git
```
2. Відкрити папку з проектом
```
cd java-university-project
```
3. Запустити додаток через maven
```
mvn clean javafx:run
```
