# java-university-project
## [Лабораторна робота 12]( https://github.com/BohdanSharubin/java-university-project/tree/lab12_db)
### Завдання
- Створіть таблицю у базі даних для вашого класу.
    - поле id — PRIMARY KEY;
    - інші поля — відповідно до атрибутів вашого класу;
    - загальна кількість полів — не менше 4.
- Додайте до проєкту файл schema.sql (або create_table.sql) з SQL-командою створення таблиці.
- Додайте файл db.properties (або app.properties) з параметрами підключення:

```
db.url=jdbc:postgresql://localhost:5432/your_db
db.user=your_user
db.password=your_password
```
- Під час запуску програми шлях до цього файлу повинен передаватися через аргументи командного рядка.

```
java Main app.properties
```
- Після створення об’єкта у програмі необхідно виконувати INSERT у таблицю.
- Для виконання INSERT використовуйте PreparedStatement. Формування SQL-запитів через конкатенацію рядків заборонено.
- Передбачте обробку помилок підключення та виконання SQL-запитів.
### Вимоги до структури проєкту
- Проєкт з попередньої лабораторної роботи;
- файл schema.sql;
- файл db.properties (або app.properties);
- окремий клас для роботи з БД (наприклад, DatabaseManager або Repository);
- читання конфігурації через args;
- коректна обробка винятків.
### Як запустити
1. Потрібно клонувати репозиторій.
2. У файлі app.properties вказати правильні дані для підключення, тобто user, password і url для підключення
3. Запустити.
```
java Main app.properties
```
